# meteor-translit
URL-safe transliteration for Meteor.

## Usage

```
translit('Съешь ещё этих мягких французских булок, да выпей чаю'); // => 'sesh-esche-etih-myagkih-francuzskih-bulok-da-vypey-chayu'

translit('Широкая электрификация южных губерний даст мощный толчок подъёму сельского хозяйства'); // => 'shirokaya-elektrifikaciya-yuzhnyh-guberniy-dast-moschnyy-tolchok-podemu-selskogo-hozyaystva'

translit('Жебракують філософи при ґанку церкви в Гадячі, ще й шатро їхнє п’яне знаємо'); // => 'zhebrakuyut-filosofi-pri-ganku-cerkvi-v-gadyachi-sche-y-shatro-yihnye-pyane-znayemo'

translit('Аффтар, зохавай исчо тех аццких олбанских креведок, да выпей йаду'); // => 'afftar-zohavay-ischo-teh-acckih-olbanskih-krevedok-da-vypey-yadu'
```
